# Portfolio Abdoul Otniel BONKOUNGOU

Projet React + Tailwind CSS prêt à publier sur Vercel.

## Lancer en local
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
```

## Publier sur Vercel
1. Dézipper ce dossier.
2. Le pousser sur GitHub.
3. Importer le dépôt dans Vercel.
4. Build command : `npm run build`.
5. Output directory : `dist`.

## Photos
Les photos sont dans `public/` :
- `photo-portrait.jpg`
- `photo-avatar.jpg`

## CV
Pour activer le bouton “Télécharger mon CV”, ajoute ton CV ici :
- `public/cv.pdf`
